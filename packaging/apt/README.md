# LoFiBox Preview APT Repository Assets

This directory contains user-facing examples for the temporary LoFiBox-maintained
APT repository hosted through GitHub Pages before LoFiBox is accepted into the
official Debian archive.

The repository must be signed and consumed through a per-repository keyring.
Do not document or use `apt-key`.
